An intelligent Full Stack Agentic AI platform that helps users automate tasks, generate content, analyze documents, and interact with powerful AI assistants. Built with modern web technologies and Large Language Models (LLMs), Zyvora delivers a fast, secure, and user-friendly experience.

# 🚀 Installation

```bash
# Clone Repository
git clone https://github.com/yourusername/zyvora.git

# Navigate
cd zyvora

# Install Frontend
cd client
npm install

# Install Backend
cd ../server
npm install

# Start Development Server
npm run dev
```
